# 晨泽发卡系统 - 成品包

## 最新版本

- **v1.0.0** - P0 完整功能版

## 下载

见 `full/` 目录下的 `chenze_faka_v1.0.0.zip`

## 包内容

- `chenze_faka` - Linux amd64 静态编译二进制（内含前端资源）
- `config.yaml.example` - 配置文件示例

## 部署

1. 解压 zip 包
2. 复制 `config.yaml.example` 为 `config.yaml` 并填写配置
3. 运行 `./chenze_faka`
4. 访问 `http://localhost:8080/install` 完成安装
